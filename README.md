# OS - Simple System in node.js

**Version** `0.0.1`,
**Author** `Mateusz#4711`

Clone repo: `git clone https://github.com/mateuszkdev/os`

## How to start

**1.** `npm i` (make shure u have typescript compiler)

**2.** `tsc --build`

**3.** `node ./build/run.js`

**4.** `You should see opening window, if no u also can go to http://127.0.0.1:3040/`
